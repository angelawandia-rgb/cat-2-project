# Zuri Café Website

A fully responsive multi-page website for **Zuri Café**, a fictional café based in Westlands, Nairobi, Kenya. Built as a Web Application Development assignment (CAT 2) using HTML5, CSS3, JavaScript ES6+, and Bootstrap 5.

## Pages

| Page | Description |
|------|-------------|
| index.html | Homepage with hero banner, highlights and CTA |
| about.html | Our story, mission, values and team |
| menu.html | Full food and drinks menu with KSh prices |
| gallery.html | Filterable photo gallery (Coffee / Food / Events) |
| blog.html | News and stories from the café |
| testimonials.html | Customer reviews and star ratings |
| contact.html | Contact form with validation and Google Maps |
| faq.html | Frequently asked questions with Bootstrap accordion |

## JavaScript Features

- **Gallery Filter** — Filter photos by category (Coffee, Food, Events) with instant show/hide
- **Form Validation** — Contact form validates name, email format, and message length before submission with a success message on valid input

## Technologies Used

- HTML5 (semantic tags throughout)
- CSS3 (custom properties, flexbox, transitions, responsive design)
- JavaScript ES6+ (event listeners, DOM manipulation)
- Bootstrap 5.3 (grid system, navbar, accordion, cards, modals)
- Git & GitHub (version control with 34+ incremental commits)
